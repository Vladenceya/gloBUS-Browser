# 🌐 gloBUS Browser

**gloBUS Browser** — это экспериментальный open-source браузер, написанный на **C# + WPF**, создаваемый как простая и понятная альтернатива привычным браузерам.
Проект ориентирован на прозрачность работы, минимализм интерфейса и полный контроль пользователя над приложением.
Цель — создать **понятный, локальный и открытый браузер**, который можно изучать, изменять и использовать без лишней сложности.

---

## ✨ О проекте
gloBUS Browser — desktop-браузер на базе **Microsoft WebView2**, где основной упор сделан на:
* простую и читаемую архитектуру
* предсказуемое поведение приложения
* открытый исходный код
* отсутствие избыточных элементов интерфейса
Проект распространяется по лицензии **MIT**, поэтому код можно свободно изучать, изменять и использовать в собственных целях.

---

## ⚙️ Возможности
### 🌍 Просмотр веб-страниц
* загрузка и отображение сайтов через WebView2
* полноценный рендеринг веб-контента
* работа как desktop-браузера
### 🔎 Адресная строка
* ввод адресов сайтов
* переход по URL
* базовая работа с навигацией
### ⬅️➡️ Навигация
* переход назад и вперёд
* обновление страницы
### 🧠 Архитектурная основа
Проект изначально построен с использованием **MVVM**, чтобы интерфейс, логика и данные были разделены.
* **View** — визуальная часть (XAML)
* **ViewModel** — состояние интерфейса
* **Services** — логика приложения
* **BrowserEngine** — изолированная работа с WebView2
Такой подход позволяет развивать проект без усложнения структуры и сохранять понятность кода.

---

### 💾 Локальное хранение данных
Приложение использует:
* SQLite
* локальное хранение состояния
Работа с данными происходит через отдельные сервисы и репозитории, без прямого взаимодействия интерфейса с базой данных.

---

## 🧱 Архитектура проекта
Тип архитектуры:
* **MVVM**
* Layered Architecture
* Modular Monolith (одно приложение с независимыми модулями)
Основной принцип взаимодействия:
```
User → View → ViewModel → Service → Data → UI Update
```
Каждый слой отвечает только за свою область ответственности.

## 🚀 Как запустить
### Требования
* Windows 10 / 11
* .NET SDK
* Microsoft WebView2 Runtime
### Установка и запуск
```bash
git clone https://github.com/Vladenceya/gloBUS-Browser.git
cd gloBUS-Browser
```
Откройте решение:
```
gloBUS.Browser.sln
```
Запустите проект через Visual Studio:
```
Run (F5)
```
После запуска откроется главное окно браузера.

## 🔮 Планы развития
В дальнейшем планируется развитие браузера и добавление новых возможностей, включая:
* систему вкладок
* настройки пользователя
* встроенную консоль команд
* инструменты приватности
* поддержку расширений
* встроенные сервисы
* боковую музыкальную панель, работающую как отдельное веб-приложение внутри интерфейса браузера
Музыкальный модуль будет открываться в боковой панели через отдельный WebView и останется независимым веб-сервисом без встроенной аудио-логики внутри приложения.
